# tick roadmap

Open-source Jira TUI (`aeswibon/tick`).

## Shipped

- **v0.1–v0.2:** Releases, CI, custom JQL, error overlay, token file, page_size, lib/cache refactor, Homebrew tap + auto-bump
- **v0.3.0:** Edit summary (`S`) and priority (`P`) from detail pane
- **v0.3.1:** Labels + sprint column/filter, doctor sprint field hints
- **v0.3.2:** Desktop notify on new issues (`notify_on_refresh`), recurring background fetch
- **v0.4.0:** Virtualized table, theme gallery + `--list-themes`
- **v0.4.1:** Edit labels (`L`), cache age in header when offline
- **v0.4.2:** Sprint/backlog move (`M`), cache age in footer
- **v0.4.3:** Configurable agile `board_id` / `boards`, doctor board list
- **v0.4.4:** Sprint view tab (5th), parallel view refresh
